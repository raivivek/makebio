makeBio
-------

Quick setup bioinformatics research projects.

Usage
-----

See `makebio --help`.

For autocompletion, add the following lines to your login file:

```
eval "$(_MAKEBIO_COMPLETE=source makebio)"
```

Or for ZSH,

```
eval "$(_MAKEBIO_COMPLETE=source_zsh makebio)"
```
